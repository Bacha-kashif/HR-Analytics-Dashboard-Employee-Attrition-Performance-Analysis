# HR-Analytics-Dashboard-Employee-Attrition-Performance-Analysis
# Project Overview
Employee attrition is a major challenge for organizations, leading to increased recruitment costs, productivity loss, and disruption in team performance. This project focuses on **analyzing employee attrition patterns and workforce trends** using an interactive **HR Analytics Dashboard built in Power BI.**

The dashboard provides actionable insights to help organizations:

*   Improve employee performance
*   Reduce attrition rates
*   Identify high-risk employee segments
*   Support strategic HR decision-making



## üéØ Business Objective

The primary goal of this project is to empower HR managers and business leaders to **reduce employee attrition** and **optimize retention strategies**. By leveraging data analytics, this project identifies patterns and risk factors across key workforce dimensions:

*   **Demographics:** Age, Gender
*   **Compensation & Role:** Salary, Job Role
*   **Background:** Education Field
*   **Stability:** Tenure
